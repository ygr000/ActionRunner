# ActionRunner

一个java程序完成的文字图画，通过设置图片路径，将路径下的图片作为字符动画播放
2020/8/6 23：22  初始版本提交 
                                                                                                                                           
java -jar ./ActionRunner-1.0-SNAPSHOT.jar /yy/tupian/shuaishou 600 b2s
运行样例 第一个参数为 图片所在父路径，第二个参数为每张图片间隔毫秒数  第三个参数为 字符密度从 密到空（b2s）还是空到密（s2b）
